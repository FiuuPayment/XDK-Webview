SwiftUI

Fiuu iOS payment module integrated via the MOLPayXDK framework for SwiftUI applications.

Repository: Mobile-XDK-Fiuu_SwiftUI

Requirements​

RequirementVersion
Xcode15+
Minimum iOS12+

Installation​

Step 1 — Add Framework​

Drag and drop MOLPayXDK.framework into your application project folder. Copy both files into the project.

Step 2 — Bridging Header​

Create a bridging header file for the MOLPay XDK Objective-C framework. Add it to Swift Compiler → Objective-C Bridging Header in Build Settings.

Add to the bridging header:

#import <MOLPayXDK/MOLPayLib.h>

Step 3 — Delegate​

Add MOLPayLibDelegate to your view controller class declaration and implement:

func transactionResult(_ result: [AnyHashable: Any]!) {
    // Handle payment result
}

Step 4 — Info.plist​

  • App Transport Security Settings → Allow Arbitrary Loads → YES
  • NSPhotoLibraryUsageDescription → Payment images
  • NSPhotoLibraryAddUsageDescription → Payment images

Start Payment​

let mp = MOLPayLib(delegate: self, andPaymentDetails: paymentRequestDict)

Show the payment UI using a .sheet modifier in SwiftUI.

Close Payment​

mp.closemolpay()
note

The host application needs to implement closing the MOLPay payment module upon receiving the final callback.
