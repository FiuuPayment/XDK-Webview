.NET MAUI

Fiuu payment gateway module for .NET MAUI Android applications.

Repository: Mobile_XDK_Fiuu_DotNetMAUI

Requirements​

RequirementVersion
Visual StudioCommunity 2022 (Windows)
Newtonsoft.JsonRequired NuGet package
Minimum Android API19+
Android targetAndroid 4.4+
info

The current .NET MAUI integration focuses on Android. Native Android file placement, manifest configuration, and activity result handling are required.

Installation​

1. Add Native Files​

In Solution Explorer, right-click your MAUI Android project → Add → Existing Item → select FiuuActivity.cs.

Copy these files into your project:

  • layout_fiuu.axml → Resources/layout/
  • menu_fiuu.xml → Resources/menu/

Click Show All Files, then right-click each file → Include In Project.

2. Android Manifest​

In project Properties → Android Manifest, check WRITE_EXTERNAL_STORAGE permission.

3. Install Newtonsoft.Json​

In Package Manager Console:

Install-Package Newtonsoft.Json

4. Override OnActivityResult​

protected override void OnActivityResult(int requestCode, Result resultCode, Intent data)
{
    base.OnActivityResult(requestCode, resultCode, data);
    if (requestCode == FiuuActivity.FiuuXDK && resultCode == Result.Ok)
    {
        Console.WriteLine("Fiuu result = " + data.GetStringExtra(FiuuActivity.FiuuTransactionResult));
        SetContentView(Resource.Layout.layout_fiuu);
        TextView tw = (TextView)FindViewById(Resource.Id.resultTV);
        tw.Text = data.GetStringExtra(FiuuActivity.FiuuTransactionResult);
    }
}

Start Payment​

Intent intent = new Intent(this, typeof(FiuuActivity));
intent.PutExtra(FiuuActivity.FiuuPaymentDetails, JsonConvert.SerializeObject(paymentDetails));
StartActivityForResult(intent, FiuuActivity.FiuuXDK);
