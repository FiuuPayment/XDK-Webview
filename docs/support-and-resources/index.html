Compliance​

Fiuu Mobile XDK supports seamless, in-page checkout and is compliant with PCI-DSS v4.0 and ISO/IEC 27001:2022 standards.

Reporting Issues​

Submit issues directly to the relevant GitHub repository, or email support@fiuu.com with:

  • Framework and SDK version
  • Platform (iOS/Android) and OS version
  • Steps to reproduce
  • Payment result JSON or error logs
